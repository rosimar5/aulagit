Explorando os Fundamentos da Web

É uma jornada empolgante pelo mundo da criação e desenvolvimento de páginas da web. Nesta, disciplina, os alunos terão a oportunidade de única de mergulhar profundamente nos conceitos de essenciais que sustentam a construção da internet moderna. Combinnando teoria e prática, esta disciplina é projetada para fornecer uma base sólida para copreender e dominar os elementos cruciais que constituem a espinha dorsal da web.

Objetivos de aprendizagem

HTML(Hypertext markup language): os alunos aprenderão a estrutura básica da web usando HTML, a linguagem fundamnetal a para criar conteúdo para online. Eles explorarão a marcação de texto, images, links e listas, entendendo como criar páginas web acessíveis e bem organizadas.

CCS(Cascading Style Sheets): Os alunos descobrirão commo dar vida ás página usando CCS, aprendendo a aplicar estilo, cores e fontes e layout para tornar o conteúdo visualmente atraente e responsivo em diferentes dispositivos.

Introdução ao javaScript: Uma introdução ao javaScript fornecerá aos alunos a capacidade de adiciona interativamente e dinamismo às suas páginas permitindo que elementos respondam as acções usuários.
